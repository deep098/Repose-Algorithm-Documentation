Overview on running this project¶

  1. Either install a Python IDE or create a Python virtual environment to install the packages required

  2. Install packages required

Setup procedure¶

  1. Create a Python Virtual Environment
    • Install virtualenv: 
      pip install virtualenv
    • Create virtualenv: 
      py -m venv <name of virtualenv>
    • Activate virtualenv: 
      .\<name of virtualenv>\Scripts\activate
    • Install requirements: 
      pip install -r requirements.txt

  2. Loading sql file into database
    • Activate virtualenv: 
      .\<name of virtualenv>\Scripts\activate
    • Create username and password in postgresql

    • Creating database 
      python manage.py dbshell 
      CREATE DATABASE <name of database>;
    • Loading dump 
      psql -d <database name> -U <username> -f <path to dump file>

    • Entering details in environment 

      Enter the following in tranquil_server\tranquil_server\.env and 
      pdf_converter\app\.env      
    DEBUG=off
    ENGINE='django.db.backends.postgresql'
    DB_NAME='<database name>'
    USER_NAME='<username>'
    PASSWORD='<password>'
    HOST_NAME='localhost'
    PORT='5432'
  3. Running the project

  4. Running the tranquil_server 
    Go to tranquil_server\ and activate the virtual environment
        python manage.py runserver
  5. Running the pdf_converter 
    Go to pdf_converter\ and activate the virtual environment
        flask run
